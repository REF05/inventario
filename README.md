# Inventario Castores

## IDE 
- Visual studio code
  
## Tecnologías
- Node.js v20.11.1
- Express @5.1.0
- MySQL 8.0 CE
- Bootstrap @5.3.2

## Instalación
1. Clona el repo  
2. `npm install`  
3. Crea la BD y ejecuta `/scripts/02_inventario_esquema.sql`  
4. Copia `.env.example` a `.env` y pon tus datos  
5. `node app.js`  
6. Abre http://localhost:3000
